# 背景音乐放这里

把 Suno 导出的 mp3 **直接放进这个文件夹**，文件名必须和下面完全一致（不用改任何代码，刷新页面就生效）：

| 文件名 | 用在哪 | 基调 |
|---|---|---|
| `bgm-title.mp3` | 标题/主菜单 | 《小小》基调，最克制的版本 |
| `bgm-prologue.mp3` | 序章（出租屋 + 中转站问答） | 《小小》基调 |
| `bgm-chapter1.mp3` | 第一章·纸燕子的最后一天 | 《小小》基调，年代怀旧感最重的一条 |
| `bgm-chapter2.mp3` | 第二章·永远差一圈的兔子 | 《小小》基调，但要"停不下来的追赶"感，不要热血 |
| `bgm-chapter3.mp3` | 第三章·纸房子的两面 | 可选，暂无对应 Suno 提示词，先留空也行 |
| `bgm-understanding.mp3` | 第一、二章通关后的"关后过渡"（母女和解那段），**这一条文件复用两处** | 《树洞》基调 |
| `bgm-ending.mp3` | 固定结局《同灯》 | 《树洞》基调的完整/丰满版；**这个文件可以不放**，不放的话结局会自动借用 `bgm-understanding.mp3` |

## 使用说明

- 缺哪个文件都没关系：代码会自动退化成一段合成的氛围垫音，保证"至少有背景声"，不会报错、不会卡住。
- 文件名大小写、连字符要完全一致；扩展名必须是 `.mp3`。
- 换音轨时会自动交叉淡入淡出（约 1.2 秒），不会有硬切的突兀感。
- 不需要在代码里 import 这些文件，也不需要告诉 Claude——放对文件名就自动生效。

对应的 Suno 生成提示词见项目对话记录里"BGM 排布方案"那部分（主题曲/序章曲/第一章曲/第二章曲/理解主题，共 5 条核心 + 可选变体）。
